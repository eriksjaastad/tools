# Integrity Warden

A suite of tools for maintaining ecosystem integrity, performing deep cleanups, and managing document indices.

## Tools

- [[integrity_warden.py|Integrity Warden]] - Core logic for verifying ecosystem health and enforcing standards.
- [[deep_cleanup.py|Deep Cleanup]] - Utility for aggressive cleanup of temporary and unnecessary files.
- [[remediate_renames.py|Remediate Renames]] - Tool for fixing broken links and metadata after file renamings.
- [[rename_indices.py|Rename Indices]] - Standardizes index file naming across projects.
